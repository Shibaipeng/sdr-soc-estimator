## 研究背景与动机

荷电状态（State of Charge, SOC）的精确估计是电池管理系统（BMS）最关键的功能之一，直接影响续航预测、充放电策略和电池安全保障。然而 SOC 无法直接测量，须通过电压、电流等间接估计，是该领域公认的研究难点。

SOC 估计方法可分为三类：

| 方法类别 | 优点 | 缺点 |
|----------|------|------|
| 安时积分法 | 简单直接 | 对初始误差和累积噪声敏感 |
| 数据驱动方法 | 无需物理模型 | 泛化能力有限，依赖大量训练数据 |
| **基于模型 + 滤波** | 精度与鲁棒性兼顾 | 模型结构决定估计上限 |

基于等效电路模型（ECM）结合卡尔曼滤波的闭环估计方法兼顾精度与鲁棒性，是工程应用的主流路线。一阶 RC（Thevenin）模型因参数少、实时性好，在车载 BMS 中应用最广。

然而，传统一阶 RC 模型存在一个**结构性缺陷**：用单一 RC 支路同时表征电化学极化（快动态，τ ≈ 10~100 s）和固相扩散浓差极化（慢动态，τ ≈ 100~1000 s），在静置阶段无法准确描述电压慢弛豫，导致 SOC 估计出现系统性偏差。

### 现有改进路线的局限

| 路线 | 方法 | 问题 |
|------|------|------|
| 增加 RC 阶数 | 二阶/三阶 ECM | 参数增多、计算负担加重 |
| 电化学/分数阶模型 | P2D 模型、分数阶微积分 | 参数多、实时性差 |

上述方案均难以满足嵌入式 BMS 的实时性约束。

---

## 本文贡献

在不增加 RC 支路、不改变模型阶数的前提下，提出一种**轻量化模型改进方案**：

1. **引入一阶惯性环节** G<sub>sd</sub>(s) = K<sub>sd</sub> / (τ<sub>sd</sub> · s + 1) 作为固相扩散修正项，仅增加 2 个可辨识参数，实现快慢动力学的**解耦表征**
2. **提出脉冲-静置两阶段参数离线辨识方法**，τ<sub>sd</sub> 和 K<sub>sd</sub> 辨识误差分别低至 0.7% 和 0.2%
3. **改进模型 SOC 估计 RMSE 从 1.23% 降至 0.60%**（降幅 50.9%），三项精度指标全部达标

| 指标 | 目标值 | 传统模型 | 改进模型 | 降幅 |
|------|:-----:|:-------:|:-------:|:----:|
| 最大误差 MaxE | ≤1.5% | 1.86% ✗ | **1.36% ✓** | 26.7% |
| 平均误差 MAE | ≤0.8% | 1.09% ✗ | **0.48% ✓** | 56.6% |
| 均方根误差 RMSE | ≤1.0% | 1.23% ✗ | **0.60% ✓** | 50.9% |

## 方法论框架

![Methodology](fig_methodology.png)

整体框架分为三个层次：

### 第一层：物理系统层
Li-ion 电池（NMC/石墨体系）提供端电压 V<sub>t</sub>、电流 I、温度 T 测量信号。OCV-SOC 特性通过三阶多项式标定。

### 第二层：模型层
- **改进等效电路模型**：在传统一阶 RC 基础上引入固相扩散修正项 V<sub>sd</sub>
- **离线参数辨识**：脉冲-静置方法获取 τ<sub>sd</sub>、K<sub>sd</sub>

### 第三层：估计层
UKF 执行 Sigma 点生成 → 状态预测 → 测量更新 → 卡尔曼增益 → 状态校正的五步递推，电压新息通过闭环反馈校正状态估计。

## 电池内部结构与固相扩散机理

![Battery Structure](fig_battery_structure.png)

锂离子电池由正极（NMC）、负极（石墨）、隔膜和电解液组成。放电时，Li⁺ 从负极石墨层间脱嵌，经电解液穿过隔膜嵌入正极 NMC 晶格，电子则通过外电路由负极流向正极，对外做功；充电过程反向。

**固相扩散慢弛豫的物理本质**：正负极活性材料均为微米级多孔颗粒，Li⁺ 在颗粒内部的固相扩散系数 D<sub>s</sub> ≈ 10⁻¹¹ ~ 10⁻¹³ cm²/s，远小于液相扩散系数 D<sub>e</sub> ≈ 10⁻⁵ ~ 10⁻⁶ cm²/s（相差 4~8 个数量级），是电池动力学的**限速步骤**。

因此固相扩散弛豫时间 τ<sub>sd</sub> ≈ 100~1000 s，远超电化学极化时间常数 τ₁ ≈ 10~100 s。传统一阶 RC 模型将两者混入同一 RC 支路，τ₁ 只能近似折中两个时间尺度，无法解耦快慢动力学差异——这正是 SOC 估计在静置阶段漂移的物理根源。

## 改进等效电路模型

### 模型结构对比

```
传统一阶 RC (Thevenin) 模型:
  ┌──── R₁ ──── C₁ ────┐
  │                     │
  R₀                    │
  │                     │
  V_OCV(SOC)           V₁ (单 RC 同时表征快慢极化)
  │                     │
  └─────────────────────┘
  Vₜ = OCV(SOC) − V₁ − I·R₀
  x = [SOC, V₁]ᵀ  (n = 2)


改进模型 (本文):
  ┌──── R₁ ──── C₁ ────┐  ← 快极化: V₁ (τ₁ ≈ 37.5 s)
  │                     │
  R₀                    ├── Vₛₔ (惯性环节, τₛₔ ≈ 280 s)  ← 慢扩散
  │                     │
  V_OCV(SOC)            │
  │                     │
  └─────────────────────┘
  Vₜ = OCV(SOC) − V₁ − Vₛₔ − I·R₀
  x = [SOC, V₁, Vₛₔ]ᵀ  (n = 3)
```

### 状态空间方程

**状态方程：**

```
SOC(k+1)  = SOC(k) − Δt · I(k) / Qₙ
                      └── 安时积分 ──┘

V₁(k+1)   = V₁(k) · e^(−Δt/τ₁)   + I(k) · R₁ · (1 − e^(−Δt/τ₁))
            └── 极化衰减 ──┘          └── 电流激励响应 ──┘

Vₛₔ(k+1)  = Vₛₔ(k) · e^(−Δt/τₛₔ) + I(k) · Kₛₔ · (1 − e^(−Δt/τₛₔ))
            └── 固相扩散慢弛豫衰减 ──┘   └── 慢扩散响应 ──┘
```

**观测方程：**

```
Vₜ = OCV(SOC) − V₁ − Vₛₔ − I · R₀
```

| 术语 | 符号 | 物理意义 | 时间尺度 |
|------|:----:|----------|:--------:|
| 欧姆极化 | I·R₀ | 电子/离子欧姆电阻 | 瞬时 |
| 电化学极化 | V₁ | 电荷转移 + SEI 膜 | τ₁ ≈ 37.5 s（快） |
| **固相扩散** | **V<sub>sd</sub>** | **Li⁺ 颗粒内部固相扩散** | **τ<sub>sd</sub> ≈ 280 s（慢）** |

### 核心创新

V₁ 表征电化学极化快动态，V<sub>sd</sub> 独立表征固相扩散慢弛豫，**两种动力学解耦**。仅增加 1 个状态变量和 2 个可辨识参数（τ<sub>sd</sub>、K<sub>sd</sub>），计算增量几乎为零，完整保留传统一阶 RC 模型实时性好的工程优势。

## UKF 状态估计算法

### 算法流程

无迹卡尔曼滤波（UKF）通过无迹变换生成 2n+1 个确定性 Sigma 点直接传播状态分布，避免扩展卡尔曼滤波（EKF）的线性化误差，对电池 OCV-SOC 强非线性系统的估计精度更高。

```
(1) Sigma 点生成    X(i) = x̂ ± √((n+λ)·P)
                         ↓
(2) 状态预测         X−(i) = f(X(i), I)
                         ↓
(3) 观测预测         ŷ(i) = h(X−(i))
                         ↓
(4) 卡尔曼增益计算   K = Pₓᵧ · Pᵧᵧ⁻¹
                         ↓
(5) 状态校正         x̂ = x̂− + K · (y − ŷ)
```

### UKF 参数配置

| 参数 | 取值 | 说明 |
|------|:----:|------|
| α | 1.0 | 分布扩展因子 |
| β | 2.0 | 高斯分布最优值 |
| κ | 0 | 次级缩放因子 |
| Q | diag(1×10⁻⁸, 1×10⁻⁶, 5×10⁻⁷) | 过程噪声协方差 |
| R | 4×10⁻⁶ | 测量噪声协方差（σ<sub>v</sub> = 2 mV） |
| Δt | 0.1 s | 采样周期 |

## 参数离线辨识

### 脉冲-静置两阶段方法

对新增参数 τ<sub>sd</sub> 和 K<sub>sd</sub>，提出仅需常规充放电设备的实用辨识方法：

| 阶段 | 操作 | 目的 |
|:----:|------|------|
| 脉冲 | 1C 恒流放电 600 s | 使 V<sub>sd</sub> 充分建立（~17.7 mV） |
| 静置 | 开路静置 1800 s | 记录端电压弛豫曲线 |

**阶段一**：利用已知 τ₁、R₁ 扣除 V₁ 贡献，得 V<sub>corrected</sub> = OCV − V<sub>t</sub> − V₁(t)

**阶段二**：对 V<sub>corrected</sub> 做单指数非线性最小二乘拟合得 τ<sub>sd</sub> 和 V<sub>sd</sub>(0)，反算 K<sub>sd</sub> = V<sub>sd</sub>(0) / [I · (1 − e^(−t<sub>pulse</sub>/τ<sub>sd</sub>))]

### 辨识精度

![Parameter Identification](param_identification.png)

| 参数 | 真值 | 辨识值 | 误差 |
|------|:----:|:-----:|:----:|
| τ<sub>sd</sub> (s) | 280.0 | 282.1 | **0.7%** |
| K<sub>sd</sub> | 0.0080 | 0.0080 | **0.2%** |
| V<sub>sd0</sub> (mV) | 17.65 | 17.58 | ~0.07 mV |

辨识质量评定为 **GOOD**。精度得益于：600 s 长脉冲充分激励慢弛豫动态、0.5 mV RMS 低噪声、以及两阶段策略对噪声的鲁棒性优于传统对数-线性回归。

## 仿真结果

### 实验工况

```
工况一：1C 放电 1500 s → 静置 900 s → 1C 充电 1500 s → 静置 900 s
         └── 快响应 ──┘     └── 慢弛豫 ──┘     └── 快响应 ──┘     └── 慢弛豫 ──┘
```

三组对比模型：(1) 传统一阶 RC + UKF（n = 2，基线）；(2) **改进模型 + UKF（n = 3）**；(3) 自适应模型 + UKF（n = 4，τ<sub>sd</sub> 在线估计，初值 300 s）。

### 三模型 SOC 估计综合对比

![Three-way Comparison](three_way_comparison.png)

### SOC 估计误差分布

![Error Distribution](fig_error_distribution.png)

改进模型误差**集中在 ±0.5% 以内**，均值更接近零、方差更小；传统模型分散至 ±2%。

### 端电压分量分解

![Voltage Decomposition](fig_voltage_decomposition.png)

三组模型的端电压估计 RMSE 均处于较低水平（0.98~1.15 mV），表明 UKF 能有效融合电压测量新息。然而相近的端电压精度并不等同 SOC 精度——传统模型 RMSE（1.23%）约为改进模型（0.60%）的**两倍**。

根本原因在于：传统模型缺少 V<sub>sd</sub> 修正项，UKF 将静置阶段固相扩散引起的电压弛豫错误映射为 SOC 修正；改进模型正确分配了电压残差的物理来源。

### 静置阶段 SOC 漂移抑制

![Rest Period Detail](rest_period_detail.png)

放电结束后 900 s 静置期间，V<sub>sd</sub> 幅值（~20 mV）虽远小于 V₁（~37.5 mV）和 I·R₀（~62.5 mV），但因 τ<sub>sd</sub>（280 s）远超 τ₁（37.5 s），在静置阶段**累积效应显著**——这正是"慢弛豫"的物理本质。

| 模型 | 静置阶段 SOC 偏差 |
|------|:---------------:|
| 传统模型 | 1.5% ~ 2%（漂移） |
| **改进模型** | **< 0.3%** |

### 自适应 UKF 性能

自适应模型（n = 4，τ<sub>sd</sub> 在线估计）实现了平均误差 0.55% 和 RMSE 0.73%（均达标），τ<sub>sd</sub> 在线收敛至 293.1 s（与真值 280 s 偏差 4.7%），验证了增广状态 UKF 框架的可行性。但最大误差 1.86% 与传统模型持平，表明联合状态-参数估计增加了滤波器自由度与不确定性，暂态稳定性不及固定参数方案。

## 讨论与局限

1. **仅基于仿真验证**：Python 平台搭建，"真实电池"由改进模型模拟，需实物电池实验确认
2. **OCV-SOC 采用固定多项式**：未考虑温度和老化影响，实际电池 OCV 随老化会偏移
3. **未在动态驾驶工况下验证**：需在 UDDS、FUDS 等瞬态工况下进一步检验泛化能力

未来工作方向：实物实验验证、温度/老化补偿机制、SOC-SOH 联合估计、动态工况泛化。
